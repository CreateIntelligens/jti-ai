# HCIoT 後台使用說明

> 適用對象：QA、測試人員、內部操作人員

## 畫面總覽

後台分成兩個主要區塊：

- **左側／上方**：對話區，可以直接和 AI 互動測試問答效果
- **右側設定**：點右上角「人物設定」按鈕開啟，包含所有設定項目

右上角還有兩個切換按鈕：

- **切換語言**（中文 / English）：切換後整個介面和 AI 都會跟著換，且會重新開始對話
- **切換主題**：深色 / 淺色

## 對話測試

左側是模擬病患使用的對話畫面。

- 選擇下方的**常用衛教主題**，再點選題目，可以快速送出問題
- 也可以直接在輸入框打字測試
- AI 的回答會附上**參考資料**（來源文件名稱），方便確認內容依據
- 右上角有**重新開始**按鈕，可清除目前對話

## 人物設定

點右上角「人物設定」開啟，共有三個分頁。

### 1. 提示詞設定（Prompt Settings）

可以調整 AI 的：

- **角色與名稱**（persona）：AI 的自我介紹方式
- **回覆規則**：禁語、語氣、格式要求等
- **回覆字數上限**

修改後按**儲存**，重新開始對話即生效。

> 若想測試不同 prompt 的效果差異，改完後記得先重新開始對話，不然 AI 仍可能沿用舊 session 的設定。

### 2. 知識庫

管理 AI 回答時查詢的文件。HCIoT 目前有兩種知識上傳模式。

#### Q&A 題庫模式

一般題庫上傳需要指定科別/主題，支援 CSV/XLSX。系統會從 CSV 的 `q` 欄位同步預設問題，並把 Q&A row 寫入 RAG。

CSV 基本格式：

| q | a | img | url |
|---|---|-----|-----|
| 什麼是高血壓？ | 血壓持續偏高的狀態... | （選填） | （選填） |
| 如何量血壓？ | 請先靜坐5分鐘... | IMG_BP_001 | https://example.com |

- `q`：問題，會同步成該 topic 的預設問題
- `a`：回答內容
- `img`：圖片 ID 或圖片檔名片段，選填
- `url`：引用網址，選填

可點「下載範例」取得範本 CSV。

#### 一般文件知識（非 Q&A）

勾選「一般文件知識（非 Q&A）」後，文件不會掛在科別/主題底下，也不會產生預設問題。這類文件會進入獨立的文件 RAG 通道，適合整篇衛教文字、Markdown、DOCX 或純文字 CSV。

一般文件模式支援：`.txt`、`.md`、`.docx`、`.csv`。

#### 手動輸入

「手動輸入」分頁可逐列輸入問題、回答、圖片。每列旁邊有「顯示」勾選框：

- 勾選：該問題顯示在前端預設問題 chips
- 取消勾選：該問題仍會上傳並進入 RAG，但不顯示在預設問題 chips

### 3. 查看和編輯文件

- 點一般文件可預覽內容；TXT、Markdown、CSV、DOCX 支援線上編輯
- 點 topic 題庫會開啟合併 CSV 編輯器
- 在合併 CSV 編輯器的編輯模式中，也可以逐列調整「顯示」勾選狀態
- 點垃圾桶可刪除文件；若某 topic 下所有 CSV 都刪完，topic 會自動移除

### 4. 科別與題目管理

這裡管理快速問答按鈕顯示的題目，以及科別和主題的名稱。

```text
科別（例如：骨科+復健科）
  └── 主題（例如：PRP 治療）
        └── 題目列表（可逐題控制是否顯示）
```

操作：

- **展開科別**：點科別名稱或左側箭頭
- **新增科別**：點最下方「＋ 新增科別」
- **新增主題**：展開科別後點「＋ 新增主題」
- **改名**：點鉛筆圖示
- **刪除**：點垃圾桶，會彈出確認視窗
- **編輯題目**：展開主題後點「編輯題目」，可修改題目文字，也可控制單題是否顯示

上傳含 `q` 欄位的 CSV 時，題目會自動同步到對應主題。同步只影響上傳當下的語言。

## 常見操作流程

### 新增一個主題並上傳 Q&A 題庫

1. 進入**人物設定 → 知識庫**
2. 科別選「＋ 新增科別」（或選現有的）
3. 主題選「＋ 新增主題」（或選現有的）
4. 上傳 CSV/XLSX 文件，或切到「手動輸入」逐列輸入
5. 需要隱藏的預設問題取消「顯示」勾選
6. 回到對話區測試

### 上傳不掛 topic 的文件知識

1. 進入**人物設定 → 知識庫**
2. 開啟新增內容視窗
3. 勾選「一般文件知識（非 Q&A）」
4. 上傳 `.txt`、`.md`、`.docx` 或純文字 `.csv`
5. 回到對話區直接提問測試

### 修改 AI 的回答風格

1. 進入**人物設定 → 提示詞設定**
2. 修改 persona 或回覆規則
3. 儲存
4. 點**重新開始**，再測試

### 測試英文版

1. 點右上角**切換語言**
2. 確認快速問答題目是否顯示英文
3. 在英文模式下上傳或編輯英文知識庫與英文題目

## 注意事項

- 修改 prompt 後，**必須重新開始對話**才會生效
- 中文和英文的**知識庫文件**、**快速問答題目**是分開管理的
- CSV 裡的問題只會更新**上傳當下那個語言**的題目，不會影響另一個語言
- 取消顯示的問題仍會進入 RAG，只是不出現在預設問題 chips
- 刪除文件後，如果該主題下所有 CSV 都刪完了，主題會自動刪除
- AI 的回答有字數限制，太長的回覆會被截斷（可在提示詞設定調整）
