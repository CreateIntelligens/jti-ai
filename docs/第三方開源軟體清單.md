## Hosted Services（非開源授權範疇）

Hosted service 條款不屬於開源授權，需另行確認各服務的商用條款。本專案實際使用的外部服務：

| 服務 | 用途 | 條款 | 備註 |
| --- | --- | --- | --- |
| Google Gemini API | LLM 推論（chat、quiz 等） | [Gemini API Additional Terms](https://ai.google.dev/gemini-api/terms) | 本專案使用付費 tier（Cloud Billing），依條款輸入資料不會被用於改進模型。若日後新增 API key，需確認同為付費方案（免費 tier 條款不同）。 |

## 商用結論

目前已確認的直接應用程式依賴多為 MIT、Apache-2.0、BSD-3-Clause、ISC、MIT-0 等寬鬆授權，通常可商用。

需注意：

- Apache-2.0、MIT、BSD、ISC 等授權仍需保留授權與 copyright notice。
- Docker base image 與 OS 套件可能包含 GPL-family 元件；GPL-family 不禁止商用，但若散佈映像檔，可能有 notices/source code 等合規義務。
- 完整 transitive dependencies 與 OS package 版本需以 release build 的 SBOM 補齊。

## 範圍與限制

本表涵蓋專案直接宣告或 Dockerfile 直接引入的第三方開源套件。完整 transitive dependencies 之逐套件清單，以 release build 產出之 SBOM 為準；如稽核需要，可另行提供：

- Backend image：Python 套件與 OS 套件 SBOM。
- Frontend：於乾淨環境執行 `pnpm install --frozen-lockfile` 後產出之 npm 套件 SBOM。
- Docker image：針對最終映像檔產出之 image-level SBOM。

前端 `pnpm-lock.yaml` 共 339 個 package entries；經對現有安裝樹之授權 metadata 掃描（111 個 package），授權分布為 MIT 100、Apache-2.0 9、MIT-0 2，未發現 GPL/AGPL/LGPL/SSPL/BUSL/PolyForm/Commons Clause 等授權。
