## 範圍與限制

- 本文件涵蓋專案直接宣告的應用程式相依套件、模型與字型資產，以及 Docker 映像中隨映像散佈的主要 OS 套件；GPL-family 套件均已標示精確版本與 SPDX 授權標示，授權全文連結列於「授權全文」表。
- 完整 transitive dependencies（含 Docker image 全部 layers）之逐套件 source URL、copyright notice 與 license 全文，以 release build 產出之 SBOM 為準；如稽核需要，可另行提供以 SBOM 工具（syft / trivy）產出的完整 notices bundle。

## 文件維護

- 若更換 `EMBEDDING_MODEL`、Docker base image 或前端 lockfile，本文件需重新查核產生。
