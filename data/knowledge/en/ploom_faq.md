# Ploom X 常見問題


## 關於 Ploom

**Q: 什麼是 Ploom？**

A: Ploom 是我們的加熱菸品牌，使用「HeatFlow™ 熱對流菸草加熱科技」，採用外部加熱 ( 而不燃燒 ) 七星 ( Mevius ) 菸彈，因此無燃燒產生的煙霧，並保有原本菸草風味。

---

**Q: Ploom X 是否比紙菸更為安全 ?**

A: 使用任何形式的菸品都有害健康，沒有一款菸草產品是絕對安全的。

---

**Q: Ploom X 是電子煙還是紙菸？**

A: 都不是。 Ploom X 是一款加熱式裝置，使用「HeatFlow™ 熱對流菸草加熱科技」，採用外部加熱 ( 而不燃燒 ) 菸彈，因此無燃燒產生的煙霧，並保有原本菸草風味。 Ploom X 需搭配專為 Ploom 所設計的的七星 ( Mevius ) 菸彈使用。

---

**Q: 什麼是 Mevius／七星菸彈？**

A: Mevius／七星菸彈，也就是 Ploom X 所使用的的菸彈。 \r\n 從原物料到最終成品，我們維持一貫地對品質的最高標準來製造 Mevius／七星菸彈。 \r\n Ploom X 的設計需搭配 Mevius／七星菸彈（簡稱菸彈）使用。 \r\n 瞭解更多有關 Mevius／七星菸彈可以查看菸彈的常見問題條目。 \r\n

A: Ploom X 使用熱流技術將菸彈加熱到最佳溫度。 \n 將您的 Mevius／七星菸彈放置在加熱器內，Ploom X 會快速高效地進行加熱。 \n

**Q: 我的菸彈裏是哪種菸草？**

A: 菸草的配方複雜且屬於公司機密，內有多種原料。

**Q: 加熱菸彈和紙菸或雪茄有什麼區別？**

A: 紙菸、雪茄和加熱菸都使用菸草 ; 區別在於紙菸與雪茄都採用燃燒菸草的方式，而加熱菸採取加熱不燃燒之方式。 Ploom X 是一款加熱式裝置，使用「HeatFlow™ 熱對流菸草加熱科技」，採用外部加熱 ( 而不燃燒 ) 菸彈，因此無燃燒產生的煙霧，並保有原本菸草風味。 Ploom X 需搭配專為 Ploom 所設計的的七星 ( Mevius ) 菸彈使用。 您無法以使用傳統菸品的方式抽專為 Ploom 設計的七星 ( Mevius ) 菸彈，也無法在 Ploom X 加熱器中置入紙菸或雪茄使用。

---

**Q: 菸彈裏含有丙二醇嗎？**

A: 是的，菸彈確實含有丙二醇。

---

**Q: 菸彈會產生焦油嗎？**

A: Mevius 菸彈和一般紙菸一樣，裡面包含焦油、尼古丁，都是在菸害防制法的規範內，而我們所有的產品都符合相關法規。

---

**Q: 可以用 Ploom X 加熱任何菸彈嗎？**

A: 不可以。 \r\n 應使用專門為 Ploom 裝置製作的 Mevius／七星菸彈。 \r\n 為其他加熱菸裝置而製作的菸彈無法正常在 Ploom X 中使用，並可能會損壞裝置。 \r\n

**Q: 我該如何處理用過的菸彈？**

A: 請小心地拿出您用過的菸彈。 \n 用過之後可能會很燙。請等到菸彈冷卻後直接丟棄。請丟棄於一般垃圾類別。 \n

## 裝置保養

**Q: 什麼是 Ploom X？**

A: Ploom X 是一款加熱式裝置，使用「HeatFlow™ 熱對流菸草加熱科技」，採用外部加熱 ( 而不燃燒 ) 七星 ( Mevius ) 菸彈，因此無燃燒產生的煙霧，並保有原本菸草風味。 Ploom X在充飽電的情況下，至少可使用20根菸彈，最多可以連續不間斷的啟動3次，且具備5分鐘不限口數之產品規格。

---

**Q: 如何使用 Ploom X？**

A: 如果這是您第一次使用 Ploom X 和 Mevius／七星菸彈，請放心，我們會協助您了解如何使用 Ploom X。 以下是如何使用 Ploom X 的說明： 1. 首次使用時，請透過原裝盒中附帶的電源供應器和 USB-C 傳輸線將裝置充飽電。 2. 滑開滑蓋。 3. 將菸彈插入加熱器。確保不要推得太深。實線應與裝置頂部對齊。 4. 按住按鈕區，直到 Ploom X 振動。您可在 LED 指示燈上查看加熱進度。 5. 當 Ploom X 準備就緒時，LED 指示燈將閃爍並伴有兩次短暫的振動。 6. 加熱完成，即可使用產品。 7. 當剩餘時間僅剩 20 秒時，Ploom X 會振動，LED 指示燈也會緩慢閃動。一旦您感覺到一次長時間的振動，吸菸時段就結束了。

---

**Q: 如何清潔 Ploom X？**

A: 使用 Ploom X 清潔棒來清潔 Ploom X，插入 Ploom X 清潔棒並轉動幾次以清除殘留物，在清潔裝置之前，請確保裝置已冷卻。 如果您想知道多久清潔一次 Ploom X，我們建議您每次用完一包 Mevius／七星菸彈時都檢查一下是否有殘留物。

---

**Q: 如何為 Ploom X 充電？**

A: 以下是為 Ploom X 充電的步驟： 1. 將包裝盒中附帶的電源供應器和 USB-C 傳輸線插入 Ploom X。 2. 如果電量耗盡，大約 110 分鐘內可充滿。 3. 您可在 LED 指示燈上查看充電進度。一旦 LED 指示燈完全點亮，表示您的 Ploom X 充電完成。 4. 當裝置準備就緒時，所有 LED 指示燈將熄滅。 想知道 Ploom X 的平均充電時間嗎？我們建議您充電至少 110 分鐘，以使 Ploom X 達到最佳性能水準。

---

**Q: Ploom X 具有防水功能嗎？**

A: 沒有。 \n 您無法在水中使用 Ploom X。 \n 如果您的 Ploom X 掉進水裏或遭淋濕了，請立即停止使用。 \n

**Q: 如果我在使用 Ploom X 時取下其前保護殼，會出現什麼情況？**

**Q: 如何設定 Ploom X 手動鎖？**

A: 您可以通過按壓按鈕區或滑動滑蓋的動作鎖定 Ploom X，這樣只有您自己才能解鎖。 以下是設置方法： 1. 將 Ploom X 連接到電源。 2. 先滑開、再關閉滑蓋。 3. 然後，按下按鈕區直到 Ploom X 振動。 4. LED 指示燈將閃爍約 20 秒。閃爍時，建立一個按壓按鈕區和/或滑動滑蓋的連續動作作為密碼。您可以使用最少 3 個動作，最多 10 個動作。 5. 滑開滑蓋並再次關閉。您的密碼現已設定完畢。

---

**Q: 如何移除 Ploom X 手動鎖？**

A: 以下是如何移除手動鎖的步驟： 1. 將 Ploom X 連接到電源。 2. 先滑開再關閉滑蓋，然後按住按鈕區直到其振動。 3. LED 指示燈將閃爍 20 秒 - LED 指示燈閃爍時請勿觸碰。 4. 20 秒後，裝置會振動，燈光將閃爍兩次。 5. 操作完成 - 您已移除手動鎖。

## 年齡驗證

**Q: 我為何需要驗證年齡？**

A: 在台灣，禁止向二十歲以下的任何人販售菸品。 \n 根據法律規定，我們需要驗證您的年齡，以確保您達到使用菸品的法定年齡。 \n 除了驗證您的年齡外，我們不會將您的個人資訊用於其他任何目的，我們將在幾分鐘內通知您是否達到了法定年齡。如需瞭解更多資訊，請查看我們的 隱私政策 。 \n

---

**Q: 我何時需要驗證年齡？**

A: 我們需要驗證您的年齡，確定您已達法定使用菸品年齡，才能瀏覽我們的網站內容。 如需瞭解有關年齡驗證的更多資訊，請查看我們的其他常見問題解答： 我為何需要驗證年齡？

---

## 帳戶註冊

**Q: 無法向 Ploom 驗證我的年齡會怎樣？**

A: 您需要驗證您的年齡才能打開 Ploom 帳戶、註冊 Ploom X。 如果我們無法線上驗證您的年齡（ 我為何需要驗證年齡？ ），請 聯絡我們 。 我們的 Ploom Care 團隊將竭誠為您服務。在我們驗證您的年齡後，您將可以開設 Ploom 帳戶。

---

**Q: 如何重設登入密碼？**

A: 如果您忘記了密碼，無法登入，我們將幫助您重置密碼。 \n 點擊登入頁面上的「重置密碼」，然後輸入您的電子郵件地址或電話號碼。我們將核對您的資料是否正確。 \n 如果核對無誤，我們將向您發送密碼重置連結。 \n 如果您未在收件箱中看到我們的電子郵件，您可能需要查看垃圾郵件文件夾。按照說明進行操作，您將能夠重置密碼。 \n 如果您仍然無法登入，請 聯絡我們 ，我們的 Ploom Care 團隊將竭誠為您服務。 \n

---

## 裝置註冊

**Q: 如何註冊 Ploom X？**

A: 以下是如何註冊 Ploom X 的步驟： 1. 購買 Ploom X 加熱器後，可以掃描店家提供的 QR Code 或是提袋上方小卡的 QR Code。 2. 您將會跳轉至建立帳戶的頁面。 3. 填寫您的個人資訊。 如果您在註冊時遇到任何問題，請 聯絡我們 ，我們的 Ploom Care 團隊將竭誠為您服務。

---

**Q: 我為何需要註冊 Ploom X？**

A: 註冊 Ploom X 後，當您需要任何的售後服務，我們將能優先為您服務。

---

**Q: 我可以註冊多個 Ploom X 裝置嗎？**

A: 是的，您可以。 \n 如果您擁有多個 Ploom X，您都可以註冊。 \n 如需查看您的註冊裝置，請參見： 我的帳戶 頁面。 \n

---

## 裝置保固與更換

**Q: Ploom X 有保固期嗎 ? 保固期是多久 ?**

A: 是的 - Ploom X 有保固。自您收到註冊裝置當日起算，您的 Ploom X 保固期為 1.5 年。 您可線上查看 保固條款 。如果您在正常使用 Ploom X 時遇到任何問題，請隨時 連絡我們 ，Ploom Care 團隊將竭誠為您服務。

---

**Q: 我如何在保固期內提出換貨？**

A: 如需在保固期間提出 Ploom X 換貨，請 聯絡我們 ，我們的 Ploom Care 團隊將竭誠為您服務。 \n 我們將與您聯絡以確定故障原因，並找出是否可以修復 Ploom X 的辦法。 \n

---

**Q: 如果我在保固期內更換了 Ploom X 裝置，我的保固期會重新計算嗎？**

A: 是的。 \r\n 若您在保固期間內因非人為因素損壞，更換了 Ploom X 裝置，需要您重新為 Ploom X 註冊，其保固期將再次重新起算。 \r\n
